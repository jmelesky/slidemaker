How to use

Space or mouse click is forward-slide.

Same with right-arrow. Left-arrow backs up.

Up-arrow and down-arrow go to the first and last slides, respectively.

Esc pulls out of the slideshow mode, and shows all the slides. Click on one to go back into slideshow mode (this is handy if you lose your place and want to search for a keyword).

How it works

make_slides.py parses this file, converting each bit to html snippets using markdown. Each snippet becomes an html div, with the 'slide' class, in the final html. The slideshow.js javascript takes all the 'slide' divs and hides them, stacks them in order, and manages input and transition (with some help from jquery).
